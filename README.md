# OperationLockdown

For best experience, use Pycharm and full screen the output.
*If ran on CodeHS, text and images may be uneven or cutoff*

For game ran on Pycharm, must have "Emulate terminal in output console" Enabled.
   - 3 Lines Top Left >> Run >> Edit Configurations >> Edit Configuration Templates >> Python >> Modify Options >> Emulate Terminal in Output Console
  This setting allows os.system('cls') to properly function, it also makes the typewriter effect smoother.

Check out the google doc organizer for game winning input, aswell as secret codes if unable to beat the riddle/hint.
Also, the organizer includes the pathing tree to see what leads to what ending.
If any bugs, please reach out.
