# OperationLockdown
This game must be ran on Pycharm with "Emulate terminal in output console" Enabled.
   - 3 Lines Top Left >> Run >> Edit Configurations >> Edit Configuration Templates >> Python >> Modify Options >> Emulate Terminal in Output Console
  This setting allows os.system('cls') to properly function, it also makes the typewriter effect smooth.


For Testing Reasons, I Suggest Commenting Out gameIntro() in the while loop on the bottom.
